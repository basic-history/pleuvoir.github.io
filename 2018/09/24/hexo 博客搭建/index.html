简单记录 github-page 搭建博客的步骤

安装博客框架

## 安装 hexo
$ npm install -g hexo-cli

## 建站
$ hexo init <folder>
$ cd <folder>
$ npm install

修改配置

关于 hexo 的大部分配置均可修改 _config.yml 完成

下载主题

hexo-theme-matery

按图索骥即可

部署

安装 hexo-deployer-git

$ npm install hexo-deployer-git --save

修改配置

deploy:
  type: git
  repo: <repository url>
  branch: [branch]
  message: [message]
### 部署到远端
$ hexo deploy
